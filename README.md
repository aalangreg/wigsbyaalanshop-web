# wigsbyaalanshop-web

Site statique pour boutique de perruques. HTML/CSS/JS vanilla, pas de build step.

## Structure

```
/
├── index.html
├── catalogue.html
├── product.html
├── about.html
├── contact.html
├── data/
│   └── products.json
├── js/
│   ├── main.js
│   ├── catalogue.js
│   ├── product.js
│   ├── home.js
│   └── order-menu.js
└── styles/
    ├── main.scss
    └── main.css
```

## Setup

Compile le SCSS :
```bash
cd styles && sass main.scss main.css
```
